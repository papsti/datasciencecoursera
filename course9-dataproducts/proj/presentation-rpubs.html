Exploring Canada's rental market over time

Irena Papst
2019-09-23

Motivation

  • want to explore how the Canadian housing rental market has changed over time

  • compare rental prices in a given geographic region

  • compare across different structure types (e.g., apartment vs. row structure) and unit types (e.g., bachelor vs. 2 bedroom)

Data

  • data downloaded from Statistics Canada on 2019-09-09 and tidied 
[1] "Number of observations: 57977"
  • average monthly rental prices measured over a long time range:
range(housing$year)

[1] 1987 2018
  • other features captured for each data point:
    • geographic region, usually a city (217 regions sampled)
    • type of structure (apartment building or row building, and size based on number of units)
    • type of unit (bachelor, 1-3 bedrooms)
